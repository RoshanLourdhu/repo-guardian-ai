\# Bug Analyzer



\## Purpose

Identify coding errors, faulty logic, and unstable patterns that may lead to failures.



\## Instructions

\- Scan all supported source files in the repository

\- Detect syntax issues, undefined variables, and improper logic

\- Highlight file name and approximate location of the issue

\- Prioritize critical bugs over minor issues

\- Provide a short explanation of why the issue is problematic



\## Output

Return a structured list of detected bugs with explanations.
